# Java 17

This module contains examples and documentation for Java 17 features.

## Contents

- **Sealed Classes**: Sealed classes finalized (no longer preview)
- **Random Generators**: Enhanced pseudo-random number generators (JEP 356)
- **Context-Specific Deserialization Filters**: Improved deserialization security

Java 17 is a Long-Term Support (LTS) release with finalized features from previous versions.
