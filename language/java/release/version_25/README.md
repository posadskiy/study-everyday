# Java 25

Examples and demonstrations of Java 25 features.

## Features

- **[JEP 506: Scoped Values](https://openjdk.org/jeps/506)** - `scopedvalues/` - Immutable data sharing between methods and threads
- **[JEP 511: Module Import Declarations](https://openjdk.org/jeps/511)** - `moduleimport/` - Import entire modules with a single statement
- **[JEP 512: Compact Source Files](https://openjdk.org/jeps/512)** - `compactfiles/` - Write Java without explicit class declarations
- **[JEP 513: Flexible Constructor Bodies](https://openjdk.org/jeps/513)** - `earlysupercall/` - Statements before `super()` calls
