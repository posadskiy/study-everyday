# Java Releases

This module contains Java version-specific features and examples.

## Contents

Examples and documentation for Java releases:
- **Version 9**: Modules (JPMS), new APIs
- **Version 10**: Local variable type inference (var)
- **Version 11**: HTTP Client, new string methods
- **Version 12**: Switch expressions (preview)
- **Version 14**: Records (preview), pattern matching
- **Version 15**: Sealed classes (preview)
- **Version 16**: Records, pattern matching finalization
- **Version 17**: Sealed classes finalization
- **Version 18**: Simple Web Server
- **Version 21**: Virtual threads, pattern matching enhancements
- **Version 22**: Unnamed variables and patterns
- **Version 23**: String templates
- **Version 24**: Stream gatherers, late barrier expansion, new encryption keys, virtual thread pinning

Each version module contains code examples demonstrating new language features and API improvements.
