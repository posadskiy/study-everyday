# Java 9

This module contains examples and documentation for Java 9 features.

## Contents

- **Modules (JPMS)**: Java Platform Module System with module-info.java
- **Compact Strings**: String storage optimization
- **Concurrency Updates**: Enhanced concurrency features
- **Convenience Factory Methods**: Factory methods for collections
- **Deprecation**: Enhanced deprecation warnings
- **JShell**: Interactive Java REPL console
- **Method Handling**: MethodHandle improvements
- **Platform Logging**: Unified JVM logging API
- **Process API**: Process management updates

Java 9 introduced the module system and many API improvements.
