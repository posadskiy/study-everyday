# Java 11

This module contains examples and documentation for Java 11 features.

## Contents

- **HTTP Client**: New standard HTTP client API
- **Local Variable Syntax for Lambda**: `var` keyword in lambda parameters
- **Nest-Based Access Control**: Improved nested class access
- **String New API**: New string methods (isBlank, lines, strip, etc.)
- **@NotNull**: Null-safety annotations

Java 11 is a Long-Term Support (LTS) release with significant API improvements.
