# Java 21

This module contains examples and documentation for Java 21 features.

## Contents

- **Pattern Matching**: Pattern matching enhancements with records and switch
- **Records**: Record classes with pattern matching examples
- **Sequenced Collections**: New sequenced collection interfaces (SequencedList, SequencedSet, SequencedMap)
- **Virtual Threads**: Virtual threads (Project Loom) for lightweight concurrency

Java 21 is a Long-Term Support (LTS) release with major concurrency and language improvements.
