# JEP 496: Module-Lattice-Based Key Encapsulation Mechanism (ML-KEM)

JEP 496 introduces the quantum-resistant Module-Lattice-Based Key Encapsulation Mechanism (ML-KEM) to the Java Platform. ML-KEM is standardized by NIST in FIPS 203 and provides secure key encapsulation using public key cryptography that is resistant to quantum computing attacks.

This enables:
- Quantum-resistant key exchange over insecure channels
- Protection against future quantum computing threats
- Standardized post-quantum cryptography implementation

# JEP 497: Module-Lattice-Based Digital Signature Algorithm (ML-DSA)

JEP 497 introduces the quantum-resistant Module-Lattice-Based Digital Signature Algorithm (ML-DSA) to the Java Platform. ML-DSA is standardized by NIST in FIPS 204 and provides digital signatures that are resistant to quantum computing attacks.

This enables:
- Quantum-resistant digital signatures for data integrity
- Authentication protection against quantum threats
- Standardized post-quantum signature implementation
