# Java

This module contains Java programming language documentation, examples, and release notes.

## Submodules

- **aspect**: AspectJ examples and aspect-oriented programming
- **core**: Core Java concepts including initialization, types, collections, and language fundamentals
- **jvm**: Java Virtual Machine concepts including garbage collection, memory model, and virtual threads
- **release**: Java version-specific features and examples
  - Version 9 through 24 with new features and improvements
