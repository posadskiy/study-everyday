# English

This module contains English language learning materials and grammar documentation.

## Contents

- **Conditionals**: Zero, first, second, and third conditional structures
- **Constructions**: Various English constructions including:
  - Action after action
  - If only
  - Linking words
  - No longer, quite
  - See it happen vs happening
  - The faster the better
  - The rich (adjective as noun)
  - To do vs doing
  - Used to
  - Verb + to infinitive
  - Whose
  - Why + -ing
- **Forms**: Identifying and adding clauses, passive voice
- **Modal Verbs**: Can and related modal verbs
- **Rules**: Grammar rules and usage guidelines

Materials for learning English grammar, constructions, and language patterns.
