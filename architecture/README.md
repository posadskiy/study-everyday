# Architecture

This module contains documentation and examples related to system architecture components and patterns.

## Submodules

- **api-gateway**: API Gateway patterns and Kubernetes ingress configurations
- **cdn**: Content Delivery Network concepts and implementation
- **kafka**: Apache Kafka event streaming platform documentation and examples
- **load-balancer**: Load balancing concepts and implementations
- **nginx**: Nginx web server configuration and usage
- **rate-limiter**: Rate limiting strategies using Nginx and Micronaut
- **tracing**: Distributed tracing with Jaeger
