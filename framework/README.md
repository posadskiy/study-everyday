# Framework

This module contains documentation and examples for various frameworks.

## Submodules

- **backend**: Backend framework examples
  - **spring**: Spring Framework examples covering Bean Post Processing, Core (scopes), Dependency Injection, and JPA (transactions)
- **frontend**: Frontend framework examples
  - **react**: React framework examples including Release 16 features
  - **react-native**: React Native development tools
    - **eslint**: ESLint configuration for React Native
    - **jest**: Jest testing framework for React Native
