# Spring

This module contains Spring Framework examples and implementations.

## Contents

- **Bean Post Processing (BPP)**: Custom bean post processors and annotations
- **Core**: Spring core concepts focusing on bean scopes (Singleton, Prototype, Request, Session, WebSocket)
- **Dependency Injection (DI)**: Dependency injection examples with performers, instruments, and poems
- **JPA**: Java Persistence API examples
  - Transaction management
  - Repository patterns
  - Entity modeling

Spring Boot application examples demonstrating various Spring features and best practices.
