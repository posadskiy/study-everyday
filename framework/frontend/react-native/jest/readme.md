# Jest for React Native

This module contains Jest testing framework documentation and examples for React Native.

## Contents

- **Testing Basics**: Common assertions and matchers
  - Null/undefined checks (`toBeNull`, `toBeUndefined`, `toBeDefined`)
  - Truthiness checks (`toBeTruthy`, `toBeFalsy`)
  - Number comparisons (`toBeGreaterThan`, `toBeLessThan`, `toBeCloseTo`)
  - String matching with regular expressions
  - Array containment checks
  - Object equality

- **Mocking**: Mocking techniques for React Native components and modules

Jest is the recommended testing framework for React Native applications, providing a comprehensive testing solution with mocking capabilities and a rich assertion library.
