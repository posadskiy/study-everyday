# ESLint for React Native

This module contains ESLint configuration and rules for React Native projects.

## Contents

- **React Rules**: ESLint rules specific to React components
  - Boolean prop naming conventions
  - Button type requirements
  - Destructuring assignment patterns
  - Component display names
  - And many more React-specific linting rules

- **React Native Rules**: ESLint rules specific to React Native
  - Unused styles detection
  - Platform-specific component separation
  - Inline styles prevention
  - Color literal management
  - Style organization best practices

ESLint helps maintain code quality and consistency in React Native applications by enforcing coding standards and catching potential errors.
