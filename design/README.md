# Design

This module contains documentation and implementations of design patterns and SOLID principles.

## Contents

- **Design Patterns**: Creational, Structural, and Behavioral design patterns
  - Implementation examples in Java
  - Pattern descriptions and use cases
- **SOLID Principles**: Object-oriented design principles
  - Single Responsibility Principle
  - Open/Closed Principle
  - Liskov Substitution Principle
  - Interface Segregation Principle
  - Dependency Inversion Principle
